# Игра-пианино
## Описание
Учебный проект, написанный на pygame, реализующий основной функционал игры-пианино
## Возможности игры
- победа и проигрыш
- смена мелодии
- отображение промахов/попадений/осталось нот
- просмотр названия нот
## Чему научился

- работать со звуками
- практика по работе с играми
- работа со шрифтом
